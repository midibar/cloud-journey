
Yoga Studio – Current System
Studio Size

Approximate activity:

~450 students per month

18 classes per week

Maximum 18 students per class

Website

The studio has a website built with WordPress.

The website mainly provides information about the studio and contact options.

Students typically contact the studio through:

WhatsApp

Email

Booking Process

A student sends a message via whatsapp or email requesting a class.
The studio Owner reads the message.
The owner records a reservation in an Excel spreadsheet.
The spreadsheet is used to track class participation and availability.

However, the operational system of the studio is still primarily handled using Excel spreadsheets.

Excel is currently used for:
- class reservations
- pass tracking
- class usage calculations

The spreadsheet contains formulas that automatically track:

- pass type
- class usage
- remaining classes

The system highlights low remaining classes and indicates when passes are finished.

All entries are manually entered by the studio owner.


Database System

The studio is currently transitioning towards using Airtable as a database system.

Airtable exists as a database structure but is not yet the primary operational tool for bookings.

Main tables include:

Students

Professors

Passes

Reservations

Attendance

Payments

However, the operational booking process is still primary handled using excel spreadsheets.

Attendance Tracking

During each class, professors use a physical paper form.

The form includes columns such as:

Student Name

Simple Pass

Double Pass

Free Pass

Resident Solo Class

Tourist Solo Class

Way of Payment

Import Paid

Import Checked

After the class, the sheet is given to the studio owner.

The owner then manually records this information in Excel.

Payments

Payments can happen in two ways:

Cash payments at the studio , via paypal, Bizum or a Revolut acc.

TPV payment links

Payments are manually recorded in Excel.

Infrastructure Observations

The studio has WiFi available.

Professors do not currently have access to the database system.

Most operational data entry is performed manually by the studio owner.
